Retrieve data from Illumina BaseSpace

Description

This tool retrieves data from the Illumina BaseSpace service. To use this tool you need:

  • 1. Use account in Illumina BaseSpace
  • 2. Access Token for Illimina BaseSpace client: bs (details below)
This tool allows you to:
  • List the data sets you have in the Illumina BaseSpace.
  • Download content of a data set as gzipped tar archive file.
  • Display content of a specific data set as a table.
  • Display detailed information of a data set

Parameters

You must define the Illumina BasesSpade API server and your personal access token for every task. The data sets are identified by their names (id code is not accepted). In the case of data set listing, the name parameter is ignored and can be left blank.

Obtaining server name and access token for Illumina BaseSpace

To create the Access Token you must have access to computer that has bs client installed. You can easily install bs client to your local computer or if you have CSC account, you can use the bs client installed in taito.csc.fi. The access token is created with bs setup command: 
bs -auth

The command prints out a link, that you should paste to a browser program to verify your authentication. After the authentication, Illumina BaseSpace configuration file is created to your home directory in the computer where you executed the bs -auth command. This configuration file contains information about the server name and access token you should use.

In Taito, you can print out the content of the configuration file with command: 

cat $HOME/.basespace/default.cfg

Note, that you need to set the access token for BassSpace only once. However you must include the access token to every BaseSpace command in Chipster, as Chipster does not store the access token information.
