Utilities / List contents of a tar file

Description

Lists the contents of a tar file.

Parameters

  • None

Details

The tool lists the contents of a tar file into a file. The file can be gzip compressed.

This list can be useful e.g. when wanting to extract only some files from a tar file. You can edit it, and use it as list file with tool "Utilities / Extract .tar or .tar.gz file".

Output

The output is a text file.
