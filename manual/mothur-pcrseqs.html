Metagenomics / Customize the reference with Mothur

Description

Customize your region of interest from the Silva reference sequence.

Parameters

  • Start (Integer) []
  • End (Integer)) []
  • Remove leading and trailing dots (yes, no) [no]

    • Details

    You can customize the Silva reference sequence with this tool if you know the starting point and ending point of your region of interest. You can choose to remove or choose to keep the leading and trailing dots.

    This tool is based on the pcr.seqs command of the Mothur package.

    Output

    The analysis output consists of the following:

    • custom.reference.fasta: A FASTA-formatted sequence file for the customized reference.
    • custom.reference.summary.tsv: Summary statistics for the sequences
