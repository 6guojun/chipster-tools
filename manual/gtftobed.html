Utilities / Convert GTF to BED

Description

Converts GTF/GFF file to BED format.

Parameters

none

Details

Converts GTF/GFF file to UCSC-styled BED format compatible with Chipster tools. The tool tries to determine the format of the input file from the header.

It should be noted that the formatting of GTF files can vary, and the tool can fail on some files.

Output

  • BED file

Reference

This tool is based on the gtf2bed tool of the ea-utils package.
