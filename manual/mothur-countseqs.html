Metagenomics / Count sequences with Mothur

Description

Counts the number of sequences represented by the representative sequence in a name file. If a group file is given, it will also provide the group count breakdown.

Parameters

-

Details

This tool is based on the count.seqs command of the Mothur package.

Output

The analysis output consists of the following:

  • counts.count_table: Count table.
  • count-summary.tsv: Summary statistics for the sequences
