RNA-seq / Compare assembled transcripts to reference annotation using Cuffcompare

Description

Cuffcompare allows you to compare the transfrags you assemble to a reference annotation.

Parameters

  • Chromosome names in my GTF files look like (chr1, 1) [1]
  • Ignore non-overlapping reference transcripts (yes, no) [no]
  • Reference organism

Details

Cufflinks GTF from the sample is matched against the reference GTF, and sample isoforms are tagged as overlapping, matching, or novel where appropriate. You can supply your own reference GTF or use one of the provided ones.

Output

The analysis output consists of the following files:

  • cuffcmp.refmap.tsv
  • cuffcmp.tmap.tsv
  • cuffcmp.combined.gtf
  • cuffcmp.loci.tsv
  • cuffcmp.stats.txt
  • cuffcmp.tracking.tsv
Please see Cufflinks documentation for details of output files.

References

This tool uses the Cufflinks package for statistical analysis. Please read the following article for more detailed information:

Trapnell C, Williams BA, Pertea G, Mortazavi A, Kwan G, van Baren MJ, Salzberg SL, Wold BJ, Pachter L. Transcript assembly and quantification by RNA-Seq reveals unannotated transcripts and isoform switching during cell differentiation. Nat Biotechnol. 2010 May;28(5):511-5.
