Read quality with MultiQC for many FASTQ files

Description

This tool runs FastQC on multiple FASTQ files, and then combines the reports using MultiQC.

Parameters

Details

Input file is a single Tar package containing the FASTQ files you want to analyze. FASTQ files can be gzipped. The Tar package itself can be gzipped, but it should be noted that if the individual FASTQ files are already compressed, compressing the Tar package will not make it any smaller.

You can make a Tar package of your FASTQ files using the Utilities tool Make a tar package.

Output

An html report with plots and tables produced by the analysis modules. Please note that you have to select the visualization option "Open in external browser" or "Open in new tab".

Reference

This tool is based on the FastQC package by Simon Andrews et al., and the MultiQC package by Philip Ewels et al.
