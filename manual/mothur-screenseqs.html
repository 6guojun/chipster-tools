Metagenomics / Screen sequences with Mothur

Description

Screen out sequences that do not fulfill certain user defined criteria.

Parameters

  • Minimum length of the sequences
  • End position
  • Start position
  • Optimize by (empty, minlength, start, end) [empty]
  • Criteria (0:100)
  • Maximum number of ambiguous bases (Integer)
  • Maximum homopolymer length (Integer)

    • Details

    Screen out sequences based on their length, ending or starting position. You can either choose the cut-off yourself (first three parameters), or use the optimization based on these. If you choose to use the optimization, remember to determine the criteria: for example 85 means that mothur will optimize the cutoff for the above chosen quality so that 85% of the sequences are kept.

    This tool is based on the screen.seqs command of the Mothur package.

    Output

    The analysis output consists of the following:

    • reads.good.groups: A two column list with the first column indicating the sequence names in the input file and the second column the group that it came from.
    • reads.trim.good.names: A list of the selected sequences and the redundant sequences corresponding to each.
    • reads.trim.unique.good.fasta: A FASTA-formatted sequence file with the screened reads.
    • summary.trim.screen.tsv: Summary statistics for the sequences
