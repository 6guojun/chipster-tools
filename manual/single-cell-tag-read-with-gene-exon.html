Tag read with gene exon

Description

This Drop-seq tool adds a BAM tag GE to reads which overlap an exon of a gene. This tag contains the gene name, as reported in the GTF genome annotation file.

For more details, please check the Drop-seq manual.

Parameters

  • Organism [other]
Chipster provides Ensembl-based GTF files for many organisms in the pull-down menu. If you would like to use your own GTF, set the organism parameter to other and check that your GTF file has been correctly assigned in the parameter panel.

Output

  • merged_tagged.bam: BAM file where reads overlapping an exon are tagged with gene name.
