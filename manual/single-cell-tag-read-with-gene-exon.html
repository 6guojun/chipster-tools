Tag read with gene exon

Description

This Drop­-seq tool adds a BAM tag “GE” onto reads when the read overlaps the exon of a gene. This tag contains the name of the gene, as reported in the annotations file.

For more details, please check the Drop-seq manual.

Parameters

  • Organism [Homo_sapiens.GRCh38.86]

Output

  • merged_tagged.bam : Gene tagged BAM
