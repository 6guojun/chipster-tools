Produce species count table and phenodata

Description

This tool generates a count table and a phenodata file, which you can use to assign samples to different experimental groups for further statistical analysis. The input for the tool is a read taxonomy assignment file that can be created with the tool Classify sequences to taxonomic units, and Mothur's counttable file.

Parameters

Cutting level for taxonomic names (0-9) [0] Rarefy counts (yes, no) [no] Produce binary table instead of counts(yes, no) [no]

Details

-

Output

The analysis output consists of the following:

  • counttable.tsv: Count table where rows are samples and columns are taxonomic groups
  • phenodata.tsv: Phenodata file allowing the user to assign samples to experimental groups
