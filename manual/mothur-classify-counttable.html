Produce species count table and phenodata

Description

This tool generates a count table and a phenodata file, which you can use to assign samples to different experimental groups for further statistical analysis. The input for the tool is a read taxonomy assignment file that can be created with the tool Classify sequences to taxonomic units, and a group file. NOTE that the tool doesn't currently take Mothur's count_table file, so the taxons are shown only for the representative sequences (unless you have classified all sequences). This will be improved in September 2018, when we will also add the Mothur pipeline for producing OTU-table (dist.seqs – cluster, using opticlust - make.shared - classify.otu).

Parameters

Cutting level for taxonomic names in count table (0-9) [0]

Details

-

Output

The analysis output consists of the following:

  • counttable.tsv: Count table where rows are samples and columns are taxonomic groups
  • phenodata.tsv: Phenodata file allowing the user to assign samples to experimental groups
